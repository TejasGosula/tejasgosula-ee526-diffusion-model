# EE526 Diffusion Model Project

This repository contains a simple implementation of a Denoising Diffusion Probabilistic Model (DDPM) for educational purposes as part of an EE526 course project.

## Files
- model.py: Neural network for noise prediction
- diffusion.py: Forward diffusion and sampling utilities
- train.py: Training loop
- sample.py: Reverse diffusion sampling
- utils.py: Helper functions
- data/: Dataset information
